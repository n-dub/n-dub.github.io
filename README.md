### Personal projects

- [Ferrum3D](https://github.com/Ferrum3D/Ferrum3D) ‒ a Vulkan graphics engine and game framework written in C++. It's an experimental engine capable of rendering meshlets without textures or materials. Only relatively low-level systems are currently implemented. The engine has an RTTI with reflection, a fiber-based job system, somewhat functional frame graph, and an async uploader.

- [Actias](https://github.com/UraniumTeam/actias) ‒ an academic project. An experimental software development kit and an OS-agnostic runtime that enables binary compatibility on desktop operating systems for dynamic libraries and executables written in C and C++.
